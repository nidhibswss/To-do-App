# To-Do App

A simple To-Do web application demonstrating basic web development concepts using **Flask**.  
It allows users to manage their tasks effectively by adding new tasks, marking tasks as completed, and deleting tasks as needed.

## Technologies Used
- Python
- Flask
- HTML/CSS

## Features
- **Add Task:** Enter a task in the input field and click the "Add" button to add it to the list.  
- **Mark Task as Completed:** Click the "check" link next to a task to mark it as completed. The task text will be crossed out.  
- **Delete Task:** Click the "delete" link next to a task to remove it from the list.
